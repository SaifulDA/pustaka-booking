# Pustaka Booking
Pustaka Booking adalah contoh web booking buku pada perpustakaan

Aplikasi ini **HANYA DIGUNAKAN UNTUK PEMBELAJARAN** dan **TIDAK DISARANKAN DIGUNAKAN** pada Web yang nyata.

##Fitur
- Login
- Register
-	Kategori Buku
- Data Buku
- Data Anggota
- Data Peminjaman
- Data Booking
- Laporan Data Buku
- laporan Peminjaman

##Hak akses
- Admin
- Pengunjung
